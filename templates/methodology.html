System Methodology

1. Dataset Description

The system is trained using a structured diabetes dataset containing patient demographic details, vital signs, clinical indicators, and family medical history.

2. Data Preprocessing

  • Handling missing and inconsistent values
  • Feature normalization and scaling
  • Encoding categorical variables

3. Feature Selection

Key features such as age, glucose level, BMI, blood pressure, and hereditary factors were selected based on medical relevance.

4. Machine Learning Model

A supervised classification model is used to predict diabetes risk. The model is trained, validated, and tested to ensure robustness and generalization.

5. Prediction & Output

The trained model predicts diabetes risk probability, which is displayed to the user along with health insights.

6. System Architecture

  • Frontend: HTML, Bootstrap, JavaScript
  • Backend: Python Flask
  • Model: Scikit-learn
