# 🩺 Type-2 Diabetes Prediction Using Machine Learning Algorithms

## 📌 Project Overview

This project focuses on predicting whether a person is likely to have Type-2 Diabetes using various Machine Learning algorithms. The system analyzes medical attributes and provides predictions to assist in early diagnosis.

---

## 🎯 Objectives

* To build a predictive model for Type-2 Diabetes
* To compare different Machine Learning algorithms
* To improve accuracy in early detection
* To provide a user-friendly web interface for predictions

---

## 🧠 Machine Learning Algorithms Used

* Random Forest
* Extra Trees
* Random Forest
* LightGBM
* CatBoost
* Stacking Ensemble
* Logistic Regression

---

## 📊 Dataset

* The dataset contains medical attributes such as:

  * Glucose level
  * Blood pressure
  * BMI (Body Mass Index)
  * Age
  * Insulin level

*(Dataset used: PIMA Indians Diabetes Dataset or similar)*
